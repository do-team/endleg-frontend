About

This game was developed as AWS concept utilizing S3 storage for hosting static website pages.

  • Pages utilizes JS/Bootstrap/Angular/JQuery
  • DynamoDB as storage for user profiles and game results.
  • Cognito as login mechanism/possible to use directly FB API.
  • Kinesis as mechanism for processing parallelization (100 lambdas max. at once)

    • Thanks to Tony Fendall and his blog! We used parts of your great example as template for our project.
